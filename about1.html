About Us

​Protorous represents the examination through the internet. This project will help the university/ Institution to evaluate the question has multiple options with one correct answer. This project contains two types of question viz. subjective and objective. Students have to write the descriptive answer for the subjective questions will be randomly generated. It would be evaluated automatically i.e. by the system.

Protorous is a web-based lockdown browser that converts any system into a safe workstation.

 

It does not allow you to move out of the test window before the test is finished. It also restricts you from opening any unauthorized website or accessing any blacklisted software/third party applications.

 

Protorous also shuts down all active applications, such as other browsers and cheating tools..
